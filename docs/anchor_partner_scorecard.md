# Anchor Partner Scorecard

**Status: draft tool.** Use this scorecard to evaluate whether a candidate organization is suitable as a **first** anchor partner for the Gazipaşa pilot. Scores are subjective and should be revisited after each conversation.

## Scoring scale

- **0** — no fit
- **1** — weak fit
- **2** — medium fit
- **3** — strong fit

## Criteria

| # | Criterion | Score (0–3) |
|---|---|---|
| 1 | Has unused or low-season housing or venue capacity | |
| 2 | Cares about Gazipaşa branding and local visibility | |
| 3 | Can benefit from international founder visibility | |
| 4 | Is comfortable with a small, time-boxed pilot | |
| 5 | Does not require large upfront bureaucracy | |
| 6 | Can introduce local challenge owners or stakeholders | |
| 7 | Can tolerate experimental / uncertain outcomes | |
| 8 | Has low reputational risk (no current scandals or disputes) | |
| 9 | Can sign simple written, non-binding terms later | |
| 10 | Does not demand unsupported claims (visas, funding, approvals) | |
| **Total** | | **/30** |

## Interpretation

- **20+** — strong first-anchor candidate. Worth a discovery call and a follow-up.
- **12–19** — possible secondary partner. Keep in the pipeline, do not lead with them.
- **Below 12** — not a first-anchor candidate. Polite hold; revisit after the first pilot.

## Notes for use

- Score honestly. Inflated scores cost weeks of wasted outreach.
- A single zero on criterion 8 (reputational risk) or 10 (unsupported claims) should be treated as a stop signal even if the total is high.
- The scorecard is an internal tool. Do not share scores with candidate partners.
